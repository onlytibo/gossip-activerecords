# README
Ruby 2.5.1
Rails 5.2.3

## Install

`git clone git@github.com:thibautgallice/gossip_activerecords.git`

`cd gossip_activerecords`

`bundle install`

`rails db:reset`

`rails db:migrate`

`rails db:seed`

`rails c`


## Run

`tp City.all`

`tp User.all`

`tp Gossip.all`

`tp Tag.all`

`tp TagGossip.all`

`tp PrivateMessage.all`
